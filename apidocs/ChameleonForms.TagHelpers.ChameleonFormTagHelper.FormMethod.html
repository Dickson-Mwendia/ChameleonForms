    Enum ChameleonFormTagHelper.FormMethod

    Form method for the tag helper. POST is the default (for when the method isn't specified).

    Namespace: ChameleonForms.TagHelpers
    Assembly: ChameleonForms.dll
    Syntax
    public enum FormMethod
    Remarks

    Nullable enums don't allow you to omit the enum type in the views hence using this instead of Microsoft.AspNetCore.Mvc.Rendering.FormMethod (where GET is the default).
